  • About the Game

    Made in Unity
    Team Size: 2
    Timeline: 4-5 months
    Worked On: Game Design, Game Narrative, Game Programming, etc.

    This project was for my undergrad capstone class. I designed and planned out the project myself, and got the opportunity to work with an amazing artist that wanted to work on a game for their own capstone project.

    As the lead on the project, I designed the short narrative behind the game, as well as many other aspects including a beliefs system that the player uses at the end of each day, the narrative and character dialogue, and part of the level design. Along with this, I did all of the programming for the game.
